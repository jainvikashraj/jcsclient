
NAME :
    create-snapshot


DESCRIPTION :
    Create Snapshot in your JCS account from the volume avaliable in your
    Jcs Account.


SYNOPSIS :
    create-snapshot
    --volume-id <value>


OPTIONS :
    --volume-id <value>
    VolumeId from which the snapshot needs to be created

EXAMPLES :
    jcs compute create-snapshot --volume-id b2ba5a18-d8cc-4223-bf62-2a866f5890bf
