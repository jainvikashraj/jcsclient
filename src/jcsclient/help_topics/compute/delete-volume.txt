
NAME :
    delete-volume


DESCRIPTION :
    Delete Volume that are currently visible under your JCS
    account. Volumes in avaliable and error state can only be
    deleted.


SYNOPSIS :
    delete-volume
    --volume-id <value>


OPTIONS :
    --volume-id
    volume-id that need to be deleted.


EXAMPLES :
    jcs compute delete-volumes --volume-id b2ba5a18-d8cc-4223-bf62-2a866f5890bf
