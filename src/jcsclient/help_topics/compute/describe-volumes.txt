
NAME :
    describe-volumes


DESCRIPTION :
    Describes Volumes that are currently visible under your JCS
    account. The volumes are listed as xxxxx-xxxxx-xxxxx-xxxxx. By default,
    this would list all volumes in your account. If you need to describe
    specific volume, you can use the --volume-ids option.


SYNOPSIS :
    describe-volumes
    [--volume-ids <values>]
    [--max-results <integer>]
    [--next-token <volume_id>]
    [--detail <True | False>]


OPTIONS :
    [--volume-ids <values>]
    List of ids which need to be described.

    [--max-results <integer>]
    Max number of results to be shown in response.
    
    [--next-token <volume_id>]
    NextToken - Id of last volume seen if max number of results
    is less than total volumes.

    [--detail <True | False>]
    Detail - by default this is true. Set to false to will supress 
    detailed response

 

EXAMPLES :
    jcs compute describe-volumes

    jcs compute describe-volumes --volume-ids f0eb9973-3f88-4612-abbc-c7d211a3ed80

    jcs compute describe-volumes --volume-ids f0eb9973-3f88-4612-abbc-c7d211a3ed80 54da80d1-fdb7-46b3-9d1a-7b9fd5286916

    jcs compute describe-volumes --next-token 54da80d1-fdb7-46b3-9d1a-7b9fd5286916

    jcs compute describe-volumes --max-results 6
