NAME
      describe-subnets - describe one or more subnets

SYNOPSIS
      describe-subnets --subnet-ids <value> 
      describe-subnets help 

DESCRIPTION
       Describes one or more of your subnets.

       Mandatory arguments: --subnet-ids (string)


OPTIONS :
       --subnet-ids specify subnet-id to be displayed 

EXAMPLES :
       jcs vpc describe-subnets --subnet-ids subnet-11112222
